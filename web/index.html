Capabilities

Everything from raw stacks to publishable numbers.

One pipeline: import, interpolate, mask, render, quantify, export. What you see rendered is exactly what gets measured.

Load anything

Per-slice folders or multi-page TIFFs, one per channel. Missing z-slices are interpolated and both channels are aligned onto a common z-range automatically.

Branch-preserving cleanup

A microglia masking pipeline isolates cells without eroding fine processes. Default thresholds, a clean enhancement pass, and a speck wipe run automatically on every load.

Vascular morphometry

Volume fraction, centreline length and density, radius from the medial-axis EDT, junctions, tortuosity, and surface-to-volume ratio — all anisotropy-aware.

Per-cell microglia analysis

Branches, process tips, junctions, total length, Sholl ramification, and soma shape — computed per cell in true 3D, each ranked and inspectable on its own.

Interactive 3D, honestly rendered

Independent volume and isosurface rendering per channel. The isosurface sits at exactly the threshold the metrics use, and the registration offset snaps to the same voxel the overlap is measured on — the picture never disagrees with the numbers.

GPU or CPU

Backends are probed at startup and verified on-device; anything unusable falls back to CPU without a crash.

Neurovascular association

The spatial relationship, quantified.

Per-cell distances aggregate into population patterns: how much of the microglia population sits in the perivascular niche, and whether processes reach toward vessels ahead of the cell body.

Every export ships a metrics_provenance.csv recording the thresholds, trim, spacing, backend and dataset that produced the numbers — reproducible by construction.

Workflow

Four steps, automatic on load.

Load one dataset or page through a multi-stack project — each is cleaned the same way before it is measured.

01

Load

Pick the red and green sources. Slices are read, gaps interpolated, channels aligned.

02

Clean

Default thresholds, the clean microglia enhancement, and a speck wipe run automatically.

03

Render

Both channels appear in 3D with independent threshold, opacity, isosurface and offset controls.

04

Quantify

Run per-cell and vascular analysis, inspect single cells, then export CSV, snapshots and meshes.
